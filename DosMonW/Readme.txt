
 DosMon protokolliert DOS-Aufrufe von 16-Bit-Windows-Applikationen
 in einer Liste. Zu Einzelheiten siehe DOSMON.HLP.

 Unter Windows NT oder OS/2 ist zu beachten, dass DosMon nicht in
 einer eigenen Windows-VM gestartet wird. DOS-Aufrufe aus einer
 anderen Windows-VM sind fÅr DosMon unsichtbar!


 Installation DosMon

   1. folgende Dateien in ein eigenes Verzeichnis kopieren.

        DOSMONW.EXE
        DOSMON.DLL
        DOSMONW.HLP
        XLISTBOX.DLL


   2. die Datei DOSMONW.INI ins Windows-Verzeichnis kopieren.
